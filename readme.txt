В версии программы на C++ была частично реализована диагностическая часть, которая анализирует обработанный код на предмет возможных синтаксических ошибок а также выводит соответствующие сообщения. При запуске программы она получает на вход файл с кодом, удаляет из него комментарии, вложенные блоки неограниченной глубины, что не содержат каких либо операций, декомпозирует обработанный код на специальные символы и их комбинации, зарезервированные и незарезервированные слова, числовые значения.
Каких либо исправлений ошибок не было реализовано, вся программа выводит лишь некоторые предусмотренные диагностические сообщения об исходном коде.
В выводимый файл выводится код с удаленными комментариями и ликвидированными пустыми вложенными блоками.
